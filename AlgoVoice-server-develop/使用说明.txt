========================================
  AlgoVoice 平台使用说明
========================================

一、首次使用
-----------
1. 双击运行 "首次安装.bat"
   - 这会安装所有必需的依赖
   - 构建前端
   - 初始化数据库
   
2. 等待安装完成（可能需要5-10分钟）

3. 看到"安装完成"后，关闭窗口


二、日常启动
-----------
1. 双击 "启动.bat"
   
2. 等待进度条到100%
   
3. 浏览器会自动打开 http://localhost:8000
   
4. 如果浏览器没有自动打开：
   - 手动打开浏览器
   - 输入 http://localhost:8000


三、登录账户
-----------
管理员账户：
  用户名: Sycamore1024
  密码: admin123


四、查看策略
-----------
登录后：
1. 点击左侧菜单 "策略制定"
2. 点击 "我的策略"
3. 看到 "量化选股策略V1"


五、停止服务器
-----------
在命令行窗口按 Ctrl+C
或直接关闭命令行窗口


六、故障排除
-----------

问题1: 点击启动.bat没反应
解决: 
- 检查是否已有服务器在运行
- 右键"启动.bat" > 以管理员身份运行

问题2: 浏览器显示空白
解决:
- 等待30秒让服务器完全启动
- 按 Ctrl+Shift+R 强制刷新
- 或使用无痕模式访问

问题3: 看不到"我的策略"
解决:
- 确认已登录 Sycamore1024 账户
- 按 Ctrl+Shift+R 强制刷新
- 清除浏览器缓存

问题4: 端口被占用
解决:
- 启动.bat 会自动清理端口
- 或手动打开任务管理器，结束 python.exe 进程


七、文件说明
-----------
首次安装.bat  - 首次使用时运行（安装依赖）
启动.bat      - 日常启动服务器
检查状态.bat  - 检查系统状态


八、技术支持
-----------
如遇到问题，请检查：
1. Python 是否已安装（Python 3.8+）
2. Node.js 是否已安装（Node 14+）
3. 网络连接是否正常
4. 防火墙是否允许 8000 端口
