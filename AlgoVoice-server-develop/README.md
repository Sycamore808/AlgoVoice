## 快速开始

### 首次使用
```bash

首次安装.bat
```

### 日常启动
```bash
# 双击运行

```

服务器会自动启动，浏览器会在8秒后自动打开 `http://localhost:8000`

## 故障排除

### 服务器无法启动
1. 确认 Python 3.8+ 已安装
2. 确认 Node.js 14+ 已安装
3. 以管理员身份运行 `首次安装.bat`

### 页面显示空白
1. 等待30秒
2. 按 Ctrl+Shift+R 强制刷新
3. 清除浏览器缓存

### 端口被占用
启动脚本会自动清理端口，耐心等待即可
