为什么用Baostock？
-----------------
√ 完全免费
√ 无需登录
√ 无API限制
√ A股历史数据完整
√ 简单易用

与Wind版本的区别：
  - Wind：专业数据，需登录，有API限制
  - Baostock：免费数据，无需登录，无限制

========================================
  快速开始（3步）
========================================

步骤1：安装Baostock
-------------------
运行：install_baostock.bat

时间：1-2分钟

步骤2：下载数据
--------------
运行：download_data.bat

时间：3-6小时（无API限制！）
数据保存：baostock_data/

步骤3：运行回测
--------------
运行：run_backtest.bat

时间：30分钟 - 2小时
结果保存：backtest_results/
